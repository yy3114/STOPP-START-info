STOPP / START 準則

提升老年人處方品質的雙向篩選工具

工具概覽

STOPP 和 START 準則是一套互補的工具，旨在協助臨床醫生為老年人（通常是 65 歲及以上）優化藥物治療。這套準則涵蓋了多個生理系統，是當今老年醫學中去處方化 (Deprescribing) 和適當處方化 (Appropriate Prescribing) 的重要參考依據。

  • STOPP (Screening Tool of Older Person's Prescriptions)：關注潛在不適當的藥物 (PIMs)，即應該停止 (STOP) 使用的藥物。
  • START (Screening Tool to Alert doctors to Right Treatment)：關注潛在藥物遺漏 (PPOs)，即應該開始 (START) 使用的藥物。

STOPP 準則：應停止的藥物

以下列舉了老年患者中常見的潛在不適當藥物 (PIMs) 類別和具體藥物範例。

中樞神經系統 (CNS) - 長效鎮靜劑

應避免用於失眠、焦慮或慢性疼痛超過 4 週，因跌倒和認知障礙風險極高。

藥物範例： 苯二氮平類藥物 (Benzodiazepines)，如 **Diazepam** (地西泮), **Flunitrazepam** (氟硝西泮) 等。

心血管系統 (CVS) - NSAIDs

用於患有心力衰竭、高血壓控制不佳或慢性腎功能衰竭的患者，會導致液體瀦留或腎功能惡化。

藥物範例： 非類固醇抗炎藥 (NSAIDs)，如 **Ibuprofen** (布洛芬), **Diclofenac** (雙氯芬酸) 等。

胃腸道系統 (GIT) - 長期 PPIs

連續使用質子泵抑制劑超過 8 週而無明確適應症，可能增加感染 (如困難梭狀芽孢桿菌) 和骨折風險。

藥物範例： 質子泵抑制劑 (PPIs)，如 **Omeprazole** (奧美拉唑), **Lansoprazole** (蘭索拉唑) 等。

腎臟系統 - 高劑量利尿劑

長期使用高劑量袢利尿劑 (Loop Diuretics) 作為治療周邊水腫的一線藥物，沒有適當的臨床指徵，可能導致脫水和電解質紊亂。

藥物範例： **Furosemide** (呋塞米), **Bumetanide** (布美他尼) 等。

START 準則：應開始的藥物

以下列舉了老年患者中常見的潛在藥物遺漏 (PPOs) 類別和具體藥物範例。

心血管系統 (CVS) - 慢性心力衰竭

患有穩定性收縮性心力衰竭，若無禁忌症，應考慮使用 ACE 抑製劑。

藥物範例： ACE 抑製劑 (ACEI)，如 **Lisinopril** (賴諾普利), **Enalapril** (依那普利)；或 ARB，如 **Losartan** (氯沙坦) 等。

中樞神經系統 (CNS) - 輕度至中度阿茲海默症

輕度至中度癡呆症患者，應考慮使用乙醯膽鹼酯酶抑製劑以延緩認知功能衰退。

藥物範例： 乙醯膽鹼酯酶抑製劑，如 **Donepezil** (多奈哌齊), **Rivastigmine** (卡巴拉汀) 等。

骨骼肌肉系統 - 骨質疏鬆

若患有骨質疏鬆或有跌倒骨折風險，應考慮處方維生素 D 和鈣補充劑。

藥物範例： 雙磷酸鹽類藥物 (Bisphosphonates)，如 **Alendronate** (阿侖膦酸鈉)；以及維生素 D / 鈣片。

內分泌系統 - 糖尿病

患有第 2 型糖尿病，HbA1c 水平超過目標範圍（通常是 7.0-8.5%），應考慮使用口服降糖藥物或胰島素。

藥物範例： **Metformin** (二甲雙胍), 磺脲類藥物 (Sulphonylureas) 等。

**注意：** 此處列出的藥物和準則僅為常見範例，實際的 STOPP/START 準則包含了數百項具體建議。在臨床實踐中，藥物調整必須基於患者的個體化狀況、共病症、預期壽命和治療目標。
